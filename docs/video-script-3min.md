# Obsidian Notes 插件 · 3 分钟分享脚本

> 建议总时长：约 3 分钟。每段括号内为参考时长，可按录制节奏微调。

---

## 【开场】0:00 - 0:25

大家好，今天用 3 分钟分享一个 Obsidian 插件：**Obsidian Notes**。

它的作用就一句话：**把你在别处记的笔记，一次性或按需同步到 Obsidian**。支持 **macOS 备忘录、Joplin、思源笔记、Notion** 四个来源，全部在设置里配好，点一下侧边栏的刷新就能按勾选依次同步。下面按来源说一下怎么用、有什么亮点。

---

## 【亮点一】0:25 - 1:00 · 一键多源刷新 + macOS 备忘录

**第一点：一个刷新按钮，管多个来源。**

- 装好插件后，侧边栏会有一个**同步图标**；点它，或者用命令「刷新（按配置更新）」。
- 在设置里可以勾选：刷新时同步 **macOS 备忘录**、**Joplin**、**思源笔记**、**Notion**。勾了谁，点刷新就按顺序执行谁，不用一个个单独点。
- **macOS 用户**：在设置里填「备忘录 App 内文件夹名称」（比如 Notes）和「Obsidian 目标文件夹」（比如「备忘录」），点刷新就会把系统备忘录里该文件夹的笔记同步到 Obsidian，图片也会一起过来。

**演示建议：** 打开设置，指一下「刷新时更新」里四个勾选项；若用 Mac，可展示备忘录文件夹名与 Obsidian 目标文件夹，再点一次刷新。

---

## 【亮点二】1:00 - 1:45 · Joplin 与思源：指定范围 + 图片一致处理

**第二点：Joplin 和思源都支持「只同步某一块」，而且图片逻辑统一。**

- **Joplin**：填数据库路径、资源目录、**要导入的笔记本名称**（只导入这个笔记本及其所有子笔记本）。导入前记得**先关闭 Joplin**。笔记会按原有层级进 Obsidian，图片和附件会复制到 vault 的图片文件夹，链接自动改成 Obsidian 能识别的格式。
- **思源笔记**：思源内核要开着，在思源里打开 API、复制 Token。在插件里填**笔记本 ID**、**要同步的路径**（比如 `/` 整本或 `/某文件夹/siyuan` 只同步那一块）。路径要按思源里的文档标题层级来，例如根文档叫「漂泊者及其影子」、下面有「siyuan」，就填 `/漂泊者及其影子/siyuan`。图片从思源 data 下的 assets 复制到 vault，和 Joplin 共用同一套「图片文件夹」配置。

**演示建议：** Joplin 展示「要导入的笔记本名称」和同步后的文件夹结构；思源展示「笔记本 ID + 要同步的路径」以及同步结果。

---

## 【亮点三】1:45 - 2:35 · Notion 多页面 + 每页可指定输出文件夹

**第三点：Notion 支持多页面，并且每页可以指定导出到不同文件夹。**

- 在 notion.so 建一个 **Integration**，把要同步的页面**分享给该 Integration**，复制 Token 填到插件里。
- **要同步的页面**是一个多行框：每行一个页面 URL 或 page_id。如果某一行后面用 **Tab 分隔**再写一个文件夹名，这一页就会导出到那个文件夹（比如 `notion` 或 `项目A/notion`），方便在 Obsidian 里分门别类。
- 子页面会递归导出，数据库会按条目导出；图片同样会落到 vault 的图片目录，链接格式和 Joplin、思源一致。

**演示建议：** 展示「要同步的页面」里两行：一行只有 URL，一行是「URL + Tab + 文件夹名」，同步后指一下两个不同输出目录。

---

## 【收尾】2:35 - 3:00 · 单独命令 + 一句话总结

- 除了「刷新（按配置更新）」之外，还可以单独执行：**同步 macOS 备忘录**、**从 Joplin 导入笔记**、**从思源笔记导入**、**从 Notion 导入**，在命令面板里搜即可。
- 小结：**Obsidian Notes** 把 macOS 备忘录、Joplin、思源、Notion 收到一个插件里，**一次配置、按需勾选、一键多源刷新**；Joplin/思源 可只同步指定笔记本或路径，Notion 可多页且每页指定输出文件夹，图片处理统一。如果你有多处笔记想汇总到 Obsidian，可以试一下。插件开源，可从简介或仓库链接获取。谢谢，下期见。
